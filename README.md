# lambda-serverless-computing
Learning serverless computing with AWS Lambda 

## Installation

Clone this repository locally.

Use the package manager [npm](https://docs.npmjs.com/about-npm/) to install dependencies.

```bash
npm install
```

## Usage

```bash
sls deploy
```
